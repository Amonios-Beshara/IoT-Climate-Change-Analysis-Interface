Accuracy

TDS Accuracy

A test plan has been done to ensure that the prototype has succeeded, which can be measured by meeting the design requirements

1- Reducing noise: The sensors were set to measure exactly 80 measurements and take the average of them through lines of regression. The sensor then will send the average from the 80 measurements to firebase for analysis.

2- Accuracy of analysis: The python script provides a method for increasing accuracy. When there is an acute deflection from the general trend, the python script will respond by neglecting such deflections.
